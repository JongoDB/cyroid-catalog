IT Procedures

Password Reset Procedure

  1. Verify user identity via phone callback
  2. Reset password in AD to temporary value
  3. Set "must change password at next logon"
  4. Communicate new password via secure channel

New Employee Onboarding

  1. Create AD account with standard naming: first initial + last name
  2. Set initial password to Welcome123
  3. Add to appropriate security groups
  4. Create email account
  5. Request VPN access if needed
Note: All new employees should change their password immediately upon first login.
