====== Network Documentation ======

===== Network Segments =====

| Segment | CIDR | Gateway | Purpose |
| Internet | 172.16.0.0/24 | 172.16.0.1 | External connectivity |
| DMZ | 172.16.1.0/24 | 172.16.1.1 | Public-facing services |
| Internal | 172.16.2.0/24 | 172.16.2.1 | Corporate infrastructure |

===== Key Systems =====

| jumpbox | 172.16.1.20 | SSH bastion | 22 |
| ftp | 172.16.1.30 | FTP server | 21 |

| wiki | 172.16.2.20 | This wiki | 80 |
| ticketing | 172.16.2.30 | IT Ticketing | 80 |

| vpn | 172.16.2.50 | VPN endpoint | 1194 |

===== Firewall Rules =====

See [[it:firewall|Firewall Configuration]] for detailed rules.

Basically:
  * Internet can only reach NAT'd services (webserver, jenkins)
  * DMZ can reach specific internal services as needed
  * Jumpbox has SSH access to internal network

===== VPN Access =====

VPN server: 172.16.0.50
Protocol: OpenVPN (UDP 1194)
Config file: Available from IT or jumpbox at /etc/openvpn/client.ovpn

To connect:
  - Get credentials from IT (usually matches AD account)
  - Download config from jumpbox
  - Connect using OpenVPN client
